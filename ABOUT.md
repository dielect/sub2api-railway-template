# About Sub2API Railway Template

## 项目概述 | Project Overview

**Sub2API Railway Template** 是一个开源的 Railway 部署模板，让你能够一键部署完整的 Sub2API 运行环境。

> **Sub2API Railway Template** is an open-source Railway deployment template that allows you to deploy a complete Sub2API environment with one click.

## 技术栈 | Tech Stack

- **Backend**: Sub2API (Go)
- **Database**: PostgreSQL
- **Cache**: Redis
- **Platform**: Railway
